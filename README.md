# Hangman Game (Python)

## Description
This is a simple text-based Hangman game written in Python.
The player guesses a word one letter at a time.

## Features
- Uses 5 predefined words
- Maximum of 6 incorrect guesses
- Console-based gameplay
- Beginner-friendly logic

## Concepts Used
- random module
- while loop
- if-else statements
- strings and lists

## How to Run
1. Install Python
2. Open terminal or command prompt
3. Run:
   python hangman.py
